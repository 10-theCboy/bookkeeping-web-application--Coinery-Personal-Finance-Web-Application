# bookkeeping-app

## How to Run

### 1. Install dependencies
npm install

### 2. Start the mock backend (Terminal 1)
npx json-server --watch db.json --port 3001

### 3. Start the Vue app (Terminal 2)
npm run dev

### 4. Open in browser
http://localhost:5173
